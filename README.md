# design.md

A browsable, premium gallery of AI-readable design systems. Each entry is a single `DESIGN.md` an AI coding agent can read to build every screen in that style. Browse a look, see it rendered live, copy the file, drop it into your project.

- **257 systems**, each card rendered live in its own colours, type and button shape
- **Filter** by category (AI, Dev Tools, Fintech, Retail, Design, and more)
- **Spec-completeness rating** computed from each file, with a transparent breakdown
- **Copy or download** any `DESIGN.md` in one click

Static site, no build step required to view. Run `python3 build.py` to regenerate `data.json` after editing the `design-md/` files.
